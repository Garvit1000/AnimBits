## Overview

The `useShimmer` hook creates a smooth shimmer/shine effect on text using pure CSS animations. Perfect for loading states, premium branding, or eye-catching headings.

<Callout type="info">
  **Pure CSS:** No JavaScript animation - uses CSS gradients and keyframes for maximum performance.
</Callout>

## Usage

### Basic Example

```tsx
import { useShimmer } from "@/lib/hooks/use-shimmer"

export function ShimmerHeading() {
  const shimmerStyle = useShimmer()
  
  return (
    <h1 style={shimmerStyle} className="text-6xl font-bold">
      Premium Feature
    </h1>
  )
}
```

### Custom Colors

```tsx
import { useShimmer } from "@/lib/hooks/use-shimmer"

export function GoldShimmer() {
  const shimmerStyle = useShimmer({
    colors: [
      "transparent",
      "rgba(255, 215, 0, 0.9)", // Gold
      "transparent"
    ],
    duration: 3,
  })
  
  return (
    <h2 style={shimmerStyle} className="text-4xl">
      Gold Membership
    </h2>
  )
}
```

## Examples

### Hero Title

```tsx
import { useShimmer } from "@/lib/hooks/use-shimmer"

export function Hero() {
  const shimmerStyle = useShimmer({ duration: 3 })
  
  return (
    <h1 
      style={shimmerStyle}
      className="text-7xl font-black bg-gradient-to-r from-blue-600 to-purple-600"
    >
      Welcome to the Future
    </h1>
  )
}
```

### Loading State

```tsx
import { useShimmer } from "@/lib/hooks/use-shimmer"

export function LoadingShimmer() {
  const shimmerStyle = useShimmer({ duration: 1.5 })
  
  return (
    <div style={shimmerStyle} className="text-2xl">
      Loading...
    </div>
  )
}
```

### Premium Badge

```tsx
import { useShimmer } from "@/lib/hooks/use-shimmer"

export function PremiumBadge() {
  const shimmerStyle = useShimmer({
    colors: [
      "rgba(255, 215, 0, 0.5)",
      "rgba(255, 255, 255, 1)",
      "rgba(255, 215, 0, 0.5)"
    ],
    duration: 2,
  })
  
  return (
    <span 
      style={shimmerStyle}
      className="px-3 py-1 text-sm font-bold"
    >
      PRO
    </span>
  )
}
```

## Combining with Gradients

Layer shimmer over gradient text:

```tsx
import { useShimmer } from "@/lib/hooks/use-shimmer"

export function GradientShimmer() {
  const shimmerStyle = useShimmer()
  
  return (
    <h1 
      style={shimmerStyle}
      className="text-6xl font-bold bg-gradient-to-r from-pink-500 via-purple-500 to-blue-500"
    >
      Gradient + Shimmer
    </h1>
  )
}
```

## Performance

- **Zero JavaScript animation** - pure CSS
- Hardware accelerated
- No re-renders
- Extremely lightweight

## Browser Support

Works in all modern browsers:
- Chrome/Edge 90+
- Firefox 88+
- Safari 14+

Requires `-webkit-background-clip` support for text effects.

## Use Cases

- Premium/Pro badges
- Loading states
- Hero titles
- Brand names
- Call-to-action buttons
- Luxury product names

<Callout>
  **Tip:** Works best on bold, large text for maximum visual impact.
