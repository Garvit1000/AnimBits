## Overview

List animation components for creating engaging entrance effects for lists, grids, and collections of items. Perfect for revealing content progressively and creating visual hierarchy.

## Available Animations

### Entrance Animations
- **Slide In** - Items slide in from a direction with stagger
- **Stagger Fade** - Items fade in sequentially with customizable delay

## Use Cases

- Product grids in e-commerce
- Blog post listings
- Feature lists
- Navigation menus
- Dashboard widgets
- Gallery items
- Search results
- Comment threads
