## Overview

Icon animation components for adding life to your UI. These animations are perfect for enhancing user feedback, drawing attention to important elements, and creating delightful micro-interactions.

## Available Animations

### Entrance Animations
- **Fade In** - Smooth fade entrance effect
- **Scale In** - Scale from small to full size
- **Pop** - Bouncy pop effect with overshoot

### Interaction Animations
- **Bounce** - One-shot bounce on interaction
- **Rotate** - 360Â° rotation on trigger
- **Shake** - Horizontal shake effect
- **Wiggle** - Rotational wiggle effect

### Continuous Animations
- **Bounce Continuous** - Continuous bouncing motion
- **Pulse** - Subtle pulsing scale effect
- **Spin** - Continuous rotation
- **Heartbeat** - Rhythmic heartbeat animation

### Special Effects
- **Morph** - Morph between different icon states
- **Stroke Draw** - Draw icon stroke by stroke
