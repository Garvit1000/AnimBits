## Installation

<Tabs items={['CLI', 'Manual']}>
  <Tab value="CLI">
    ```bash
    npx shadcn add https://animbits.dev/r/animations/cards/parallax-tilt
    ```
  </Tab>
  <Tab value="Manual">
    Requires the `useParallaxTilt` hook from `@/lib/hooks/use-parallax-tilt`.
  </Tab>
</Tabs>

## Usage

```tsx
import { CardParallaxTilt } from "@/components/animations/cards/parallax-tilt"

<CardParallaxTilt className="p-6 bg-white rounded-lg border">
  <h3>Card Title</h3>
  <p>Interactive 3D card</p>
</CardParallaxTilt>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `maxTilt` | `number` | `15` | Maximum tilt angle in degrees |
| `shadowIntensity` | `number` | `0.3` | Shadow intensity (0 to 1) |
| `duration` | `number` | `0.3` | Animation duration in seconds |
| `className` | `string` | `undefined` | Additional CSS classes |
