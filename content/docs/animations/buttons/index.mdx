## Overview

Button components with built-in animations. These are semantic components that use the primitive hooks internally, providing ready-to-use animated buttons for common interactions.

<Callout type="info">
  **Built on primitives:** All button components use animation hooks internally. You can use those hooks directly on any element for maximum flexibility.
</Callout>

## Quick Start

All button components follow the same pattern:

```tsx
import { LiftButton } from "@/components/animations/buttons/lift"

<LiftButton className="px-6 py-3 bg-blue-500 text-white rounded-lg">
  Click me
</LiftButton>
```

## Customization

All button components:
- Accept standard button props
- Support Tailwind CSS classes
- Can be styled with any design system
- Work with click handlers and form submissions

## Using Hooks Directly

For maximum flexibility, use the underlying hooks on any element:

```tsx
import { motion } from "framer-motion"
import { useHoverLift } from "@/lib/hooks"

const liftProps = useHoverLift()

<motion.div {...liftProps}>
  Any element with lift effect
</motion.div>
```
