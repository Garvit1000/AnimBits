## Overview

The `BorderGradientButton` creates an animated gradient border that continuously traces around the button's perimeter, creating a premium, eye-catching effect.

    ```tsx title="components/animations/buttons/border-gradient.tsx"
    "use client"    import * as React from "react"    import { motion, HTMLMotionProps } from "framer-motion"    interface BorderGradientButtonProps extends Omit<HTMLMotionProps<"button">, "children"> {      children: React.ReactNode      duration?: number      colors?: string[]    }    function cn(...classes: (string | undefined)[]) {      return classes.filter(Boolean).join(' ')    }    export function BorderGradientButton({      children,      className,      duration = 2,      colors = ["#3b82f6", "#8b5cf6", "#ec4899"],      ...props    }: BorderGradientButtonProps) {      const buttonRef = React.useRef<HTMLButtonElement>(null)      const [dimensions, setDimensions] = React.useState({         width: 0,         height: 0,         radius: 0,        perimeter: 0       })      const gradientId = React.useId()      React.useEffect(() => {        if (!buttonRef.current) return        const updateDimensions = () => {          if (buttonRef.current) {            const styles = window.getComputedStyle(buttonRef.current)            const width = buttonRef.current.offsetWidth            const height = buttonRef.current.offsetHeight            const borderRadius = parseFloat(styles.borderTopLeftRadius) ||                                parseFloat(styles.borderRadius) || 0            const maxRadius = Math.min(width, height) / 2            const actualRadius = Math.min(borderRadius, maxRadius)            const straightEdges = 2 * ((width - 2 * actualRadius) + (height - 2 * actualRadius))            const curvedEdges = 2 * Math.PI * actualRadius            const perimeter = straightEdges + curvedEdges            setDimensions({              width,              height,              radius: actualRadius,              perimeter: perimeter || 1,             })          }        }        updateDimensions()        const observer = new ResizeObserver(updateDimensions)        if (buttonRef.current) {          observer.observe(buttonRef.current)        }        return () => observer.disconnect()      }, [])      return (        <motion.button          ref={buttonRef}          className={cn("relative", className)}          {...props}        >          {}          {dimensions.width > 0 && (            <svg              className="absolute inset-0 pointer-events-none"              width={dimensions.width}              height={dimensions.height}            >              <defs>                <linearGradient id={gradientId} x1="0%" y1="0%" x2="100%" y2="100%">                  {colors.map((color, i) => (                    <stop                      key={i}                      offset={`${(i / (colors.length - 1)) * 100}%`}                      stopColor={color}                    />                  ))}                </linearGradient>              </defs>              <motion.rect                x="1"                y="1"                width={dimensions.width - 2}                height={dimensions.height - 2}                rx={dimensions.radius}                fill="none"                stroke={`url(#${gradientId})`}                strokeWidth="2"                strokeDasharray={dimensions.perimeter}                strokeDashoffset={0}                animate={{ strokeDashoffset: [0, -dimensions.perimeter] }}                transition={{                  duration,                  ease: "linear",                  repeat: Infinity,                }}                style={{                  strokeLinecap: "round",                }}              />            </svg>          )}          <span className="relative z-10">{children}</span>        </motion.button>      )    }    ```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `duration` | `number` | `2` | Animation duration in seconds for one complete cycle |
| `colors` | `string[]` | `["#3b82f6", "#8b5cf6", "#ec4899"]` | Array of hex colors for the gradient |
| `children` | `React.ReactNode` | Required | Button content |
| `className` | `string` | `undefined` | Additional CSS classes |

All other props from `motion.button` are supported and passed through.

## Animation Details

The border gradient animation works by:

1. Using a rotatingparate animated divs for each side (top, right, bottom, left)
2. Each div moves along its respective edge with a sequential delay
3. The gradient flows smoothly as it traces around the perimeter
4. Animation runs continuously and infinitely
5. Each side's animation is perfectly timed to create a seamless loop

## Accessibility

- Maintains all standard button semantics
- Keyboard accessible (can be focused and activated)
- Animation respects `prefers-reduced-motion` when using Framer Motion's default settings
- Supports all ARIA attributes through prop spreading

## Browser Support

- Works in all modern browsers that support CSS gradients and transforms
- Requires Framer Motion for animations
- Gracefully degrades in browsers without animation support
