## Usage

```tsx
import { TextWordCarousel } from "@/components/animations/text/word-carousel"

<TextWordCarousel words={["Fast", "Modern", "Beautiful"]} />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `words` | `string[]` | Required | Array of words to cycle through |
| `interval` | `number` | `2` | Time between words (seconds) |
| `duration` | `number` | `0.3` | Transition duration (seconds) |
