## Overview

The `TextShimmer` creates a smooth gradient shimmer effect using pure CSS for 60fps performance.

## Installation

<Tabs items={['CLI', 'Manual']}>
  <Tab value="CLI">
    ```bash
    npx shadcn add https://animbits.dev/r/animations/text/shimmer
    ```
  </Tab>
  <Tab value="Manual">
    Requires the `useShimmer` hook from `@/lib/hooks`.
  </Tab>
</Tabs>

## Usage

```tsx
import { TextShimmer } from "@/components/animations/text/shimmer"

<TextShimmer>Shimmer Text</TextShimmer>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `string` | Required | Text to display |
| `duration` | `number` | `2` | Animation duration (seconds) |
| `colors` | `string[]` | `["transparent", "rgba(255,255,255,0.8)", "transparent"]` | Shimmer colors |
| `as` | `string` | `"p"` | HTML element to render |
