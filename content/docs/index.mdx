# Introduction

AnimBits is a collection of production-ready animation components built with Framer Motion and React. Copy and paste components directly into your project.

## Features

### Beautiful Animations
Carefully crafted animations for buttons, cards, text, icons, and more.

### Performance First
Optimized for smooth 60fps animations with minimal bundle size.

### TypeScript Support
Fully typed components and hooks for better developer experience.

### Customizable
Easy to customize with Tailwind CSS and component props.

### Tree Shakeable
Import only what you need. No bloated dependencies.

### Composable Hooks
Reusable animation hooks that work with any component.

## Quick Start

Install dependencies and add your first animation component:

```bash
npm install framer-motion clsx tailwind-merge
```

## Component Categories

Browse our collection of animation components organized by category:

- **Buttons** - Interactive button animations and effects
- **Cards** - Card hover effects and transitions
- **Text** - Text reveal and animation effects
- **Icons** - Icon animations and transitions
- **Loaders** - Loading spinners and progress indicators
- **Lists** - List item animations and stagger effects
- **Pages** - Page transition effects
