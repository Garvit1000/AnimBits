# Components

Explore our comprehensive collection of animation components built with Framer Motion and React.

## Button Animations

Interactive button effects that enhance user experience.

### Available Components

- **Ripple** - Material Design ripple effect on click
- **Magnetic** - Button follows cursor with magnetic attraction
- **Glow** - Animated glow effect on hover
- **Lift** - 3D lift effect with shadow
- **Press** - Press down animation on click
- **Border Gradient** - Animated gradient border

## Card Animations

Engaging card effects for showcasing content.

### Available Components

- **Hover Lift** - Card lifts on hover with shadow
- **Scale In** - Card scales up smoothly
- **Fade In** - Card fades in from opacity
- **Frosted Glass** - Glassmorphism effect
- **Background Zoom** - Background image zooms on hover
- **Parallax Tilt** - 3D tilt effect following cursor

## Text Animations

Dynamic text effects for headings and content.

### Available Components

- **Shimmer** - Shimmering gradient text effect
- **Scramble** - Text scramble reveal animation
- **Split Reveal** - Character-by-character reveal
- **Word Carousel** - Rotating word carousel

## Icon Animations

Animated icons for better visual feedback.

### Available Components

- **Bounce** - Bouncing icon animation
- **Rotate** - Smooth rotation effect
- **Pulse** - Pulsing scale animation
- **Stroke Draw** - SVG stroke drawing animation

## Loader Animations

Loading indicators and progress animations.

### Available Components

- **Morphing** - Shape morphing loader
- **Orbit** - Orbiting dots loader
- **Gooey Blobs** - Liquid blob animation

## List Animations

Stagger animations for list items.

### Available Components

- **Slide In** - Items slide in sequentially
- **Stagger Fade** - Staggered fade-in effect

## Page Transitions

Smooth transitions between pages.

### Available Components

- **Page Transition** - Full page fade transition
- **Slide Transition** - Sliding page transition

## Usage

Each component can be copied directly into your project. Visit individual component pages for detailed documentation, live previews, and code examples.
